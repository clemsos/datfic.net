Stalker

Retrouver qui est qui en fouillant dans les affaires personnelles.

Description

Stalker est un jeu de cartes qui joue avec nos données personnelles. Ici, tout est permis, il faut récupérer le plus d'informations sur ces adversaire. 7 Joueurs minimum.

La triangulisation des données

Le jeu stalker met en avant une problématique forte sur internet : l'anonymat. Cette dernière est quasi impossible car il suffit de quelques données pour retrouver une personne. Le jeu met au défi les joueurs d'être véritablement anonyme.

La récolte de données

Au tout début de la partie, chaque joueurs insères des informations personnelles dans une base de données. Lorsque la partie commencer, toutes les données sont mélangées. Il va falloir faire le tri et retrouver qui est qui !

le gagnant est...

Si vous récoltez suffisament de données sur un autre joueur, on stoppe la partie. Vous devez donner quatres données récoltés sur la personne, si c'est correct, le joueur est éliminé. Si c'est faux, vous êtes éliminés ! La partie continue jusqu'à n'avoir plus qu'un seul joueur... il deviens alors le gagnant de la partie !

Feedback

La réalisation de ce travail fut très intéressante. Les retours sont positifs. J'ai appris que l'anonymat sur internet est quasi impossible, qu'avec quelques données on peux retrouver une personne assez facilement.
